# Credit Risk Prediction System

This project predicts whether a loan applicant is a Good or Bad credit risk using Machine Learning.

## Tech Stack
- Python
- Scikit-learn (Extra Trees / XGBoost)
- Pandas, NumPy
- Streamlit (Web App)

## Features Used
Age, Job, Housing, Saving Accounts, Checking Account, Credit Amount, Duration.

## Model Performance
Accuracy: ~64%

## How to Run
pip install -r requirements.txt
streamlit run app.py
